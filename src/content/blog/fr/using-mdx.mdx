Ce thème est livré avec l'intégration [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) installée et configurée dans votre fichier de configuration `astro.config.mjs`. Si vous préférez ne pas utiliser MDX, vous pouvez désactiver le support en supprimant l'intégration de votre fichier de configuration.
